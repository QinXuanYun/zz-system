## System Features

### 1. Dynamic Threshold Warning (动态阈值预警)
- Warning colors: RED(预警), YELLOW(警告), BLUE(正常但有负向波动), GREEN(安全)
- 7 indicators to monitor with thresholds:
  - 报到率: Green >95%, Yellow 85-95%, Red <85%
  - 生师比: Green <20, Yellow 20-25, Red >25
  - 课程优良率: Green >85%, Yellow 70-85%, Red <70%
  - 技能证书通过率: Green >75%, Yellow 60-75%, Red <60%
  - 毕业率: Green >90%, Yellow 80-90%, Red <80%
  - 年终就业率: Green >90%, Yellow 80-90%, Red <80%
  - 专业相关度: Green >75%, Yellow 60-75%, Red <60%

### 2. Dashboard (驾驶舱)
- Show overall health status for each major with colored indicators
- Show count of green/blue/yellow/red indicators per major
- Click indicator to see details
- Summary stats: 平均就业率96.0%, 平均专业相关度69.3%

### 3. Data Visualization (数据可视化)
- Radar chart comparing multiple majors
- Bar chart for scores comparison
- Line charts showing trends (if multi-year data available)
- Tables showing detailed scores

### 4. Trend Prediction & Anomaly Detection
- Calculate averages and detect anomalies
- Z-score for detecting significant changes

### 5. Auto Report Generation (报告自动生成)
- Generate text analysis for each major
- Show strengths and weaknesses

## Technical Requirements
- Backend: Python FastAPI with these endpoints:
  - GET / - Serve frontend HTML
  - GET /api/majors - List all majors with scores
  - GET /api/indicators/{major} - Get detailed indicators
  - GET /api/warnings - Get warning status for all majors
  - GET /api/radar - Data for radar chart comparison
  - GET /api/report/{major} - Generate analysis report

- Frontend: Single page HTML with:
  - ECharts for visualizations
  - Responsive dashboard layout
  - Chinese language

- Data: JSON file with all indicator data

## Design
- Professional dashboard with dark theme (深蓝色背景)
- Card-based layout for each major
- Color coding: Green(#52c41a), Blue(#1890ff), Yellow(#faad14), Red(#f5222d)
- Clean, modern appearance
